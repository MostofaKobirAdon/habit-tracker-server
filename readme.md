# Habit-Tracker Server

A simple backend server built with **Node.js**, **Express.js**, and **MongoDB** for managing habits, streaks, and daily progress tracking.

---

## Features

- **Create new habits**
- **Get all habits**
- **Get habits by user**
- **Update habit details**
- **Delete habits**
- **Mark habit as complete (once per day)**
- **Track daily streaks**
- **Prevent duplicate completions per day**
- **Public habits endpoint**
- **Search and filter habits**
- **Category-based filtering**
- **Pagination support**
- **Includes creator name in responses**
- **Clean error handling and validation**
